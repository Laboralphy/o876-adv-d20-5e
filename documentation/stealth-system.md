# stealth

Le roublard R souhaite passer un mode furtif.

enterStealthMode
- Si une creature C a pour cible R, alors C est immédiatement noté dans la liste des créatures ayant détecté R.

R attaque une cible C
- R effectue une attaque sournoise.
- La furtivité prend fin : exitStealthMode

R change de pièce
- Les créatures dans la pièce font des jets de perception pour détecter R. Toutes les créatures sont notées de man

R est attaqué avec succès
- La furtivité prend fin : exitStealthMode
